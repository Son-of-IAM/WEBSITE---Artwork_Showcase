A high-performance, serverless portfolio and commercial hub engineered for Nigerian creative and digital artist, Son_of_IAM_ (Promise Adebayo). 

This platform was built from the ground up to showcase high-resolution 3D renders and digital paintings while maintaining an ultra-fast, app-like user experience with zero backend hosting costs.

## ⚙️ Core Architecture & Features

* **Headless Content Management:** Integrated with Decap CMS, allowing the artist to manage the gallery database (`data/artworks/index.json`) via a secure `/admin` portal without touching code.
* **Cinematic 3D Landing Experience:** A custom, CSS-driven 3D carousel that dynamically pulls the newest category representatives from the CMS database and auto-rotates in the DOM.
* **Serverless Form Processing:** All commercial inquiries, testimonies, and waitlists are handled invisibly via AJAX bypassing Formspree payload screens to ensure seamless UX and instant redirects.
* **Automated Image Pipeline:** Configured with GitHub Actions (`optimize-images.yml`) to automatically compress high-res `.jpg` and `.png` uploads on the fly.
* **Integrated Analytics & SEO:** Fully configured with Google Analytics 4 (GA4) and custom Open Graph meta tags mapped to `sonofiam.com` for premium social sharing previews.
